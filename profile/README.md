# PodPick Guide - Public Vape Data Organization

[![Website](https://img.shields.io/badge/Website-podpickguide.com-blue)](https://podpickguide.com)
[![Data](https://img.shields.io/badge/Data-Public%20Database-green)](#repositories)

> GitHub Organization for **public vape flavour and buying guide databases**.
> Not code projects - structured data for Google, Bing and AI search.

## What We Publish

| Type | Description |
|------|-------------|
| Flavour Index | Sweetness, cooling, beginner ratings per flavour |
| Rankings | Scene-based TOP lists (beginner, summer, daily carry) |
| Comparison Tables | Brand specs: Battery, Charging, Coil, Weight, Compatibility |
| FAQ Database | Structured Q&A for buying and troubleshooting |
| Buying Guides | 18+ guide index with metadata |
